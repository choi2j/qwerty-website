QWERTY가 무슨 의미죠?

저희는 키보드의 기초인 QWERTY처럼 기초에 충실하고
여러분들의 능력을 키워줄 수 있는 웹 개발 동아리 입니다.

여기선 무엇을 배울 수 있나요?

Web 기초

웹에 입문하기 위해 기초적인 구조와 지식을 공부합니다.

HTML, CSS

JS

앞으로 가장 많이 쓸 웹에서 가장 중요한 언어를 공부합니다.

JS

Vue

카카오에서도 사용하는 대중적인 웹 프레임워크, Vue를 이용해 웹 페이지를 제작해봅니다.

Vue.js

Node.js express

Node.js로 서버를 만들어보며 백엔드의 기초를 공부합니다.

Node.js, express

웹 디자인

Figma를 이용해서 더욱 멋진 웹 페이지를 만드는 법을 배웁니다.

Figma

팀 프로젝트

팀을 만들어서 기획, 디자인, 개발까지 모든 과정을 통해 완전한 웹 개발을 합니다.

자유

blender Three.js(선택)

blender로 3D 모델을 제작하고 Three.js를 통해 웹에서 모델을 보여줍니다.

blender, Three.js

SvelteKit(선택)

SvelteKit을 이용해 풀스택 프레임워크를 체험하고 웹 페이지를 제작해봅니다.

Svelte, SvelteKit
